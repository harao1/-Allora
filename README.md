配達管理システム

これは、Google Maps API を活用した簡易的な配達管理支援システムです。配達先の位置を登録したり、現在地を追跡・表示して、配達ステータス（配達完了・不在）を記録できます。
機能一覧
Google Map 上で現在地の表示
任意の住所を検索してピンを立てる
地図をダブルクリックして配達先を登録
配達完了でピンを削除
不在時にピンの色をグレーに変更
現在地の追跡（startWatchingLocation / stopWatchingLocation）
使用技術
HTML5
CSS3
JavaScript (Vanilla JS)
Google Maps JavaScript API
